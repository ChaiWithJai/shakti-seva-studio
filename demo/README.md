# Demo production

The demo is a Remotion project. It uses the real Day 0 screenshots, a plan
reviewed by local Bonsai, and narration generated by Liquid LFM2.5 Audio.

## Build the video

```bash
cd demo
npm install
npm run render
```

The checked in narration lets you render without downloading an audio model.
To make the narration again, create an isolated Python 3.12 environment and run
the generator on the CPU:

```bash
uv venv --python 3.12 .venv-audio
uv pip install --python .venv-audio/bin/python liquid-audio
.venv-audio/bin/python generate_narration.py
```

`liquid-audio` 1.3.0 selects CUDA inside its lazy detokenizer even when the
processor is configured for CPU. The generator contains a narrow construction
workaround for macOS and restores PyTorch's normal method immediately after
decoding.

The model weights use the LFM Open License 1.0. Review that license before you
redistribute weights. This repository includes only generated narration, not
model weights.

See [the Bonsai review record](bonsai-review.md) for the local model details and
the claims that were rejected during review.

## Verify the output

The accepted narration was generated in six short turns. A local Whisper
transcription covered the full script. Check the rendered streams and duration:

```bash
ffprobe -v error -show_entries format=duration:stream=codec_name,width,height,sample_rate,channels -of json ../docs/assets/shaki-seva-day0.mp4
```
