Why Shakti is here

The public demo proves when not to use AI.

Shakti reads public NYC housing repair records. Its live lookup uses ordinary code because address matching, record counts, privacy treatment, and public next steps should stay deterministic.

The optional local edition gives AI a smaller job. It explains a finished, treated packet and exposes the tool trace. It cannot change the public record.
